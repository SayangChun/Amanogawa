# AI 创作 · 天之川沙夜

把你用 AI 生成的沙夜图片放到这个文件夹即可。

## 支持格式

`.jpg` `.jpeg` `.png` `.webp` `.gif`

## 本地预览

1. 把图片丢进本目录
2. 运行 `npm start`
3. 打开 `http://localhost:3000/gallery.html`
4. 切到 **AI 创作** 分类

文件名会自动变成标题（例如 `night-saya.png` → `night saya`）。  
若要自定义标题/说明，可编辑 `src/data/ai-gallery.js`。

## 命名建议

- `saya-starry-night.png`
- `saya-school-uniform-01.jpg`
- `saya-winter-coat.webp`
